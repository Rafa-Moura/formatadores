Parâmetro Descrição Formatação
Parâmetro Monetário/Double Valores do tipo R$ ou com ponto flutuante
Parâmetro Inteiro Valores do tipo inteiro aceitando números negativos
Parâmetro Percentual Valores do tipo percentual com casa decimal
Parâmetro Percentual Inteiro Valores do tipo percentual inteiro
Parâmetro String Valores do tipo String, cadeias de caracteres
Parâmetro Data Valores do tipo Data, padrão DD/MM/YYYY
Parâmetro Hora Valores do tipo hora, padrão HH:MM:SS
Parâmetro Charset Valores do tipo Charset, aceitam um único caractere
